FFmpeg capture supervisor

Spawns FFmpeg with kmsgrab → VAAPI H.264 → RTP output to localhost. Monitors the process and auto-restarts when Gamescope changes the DRM framebuffer format (which kills kmsgrab).

Future v2: Replace FFmpeg with direct DRM/KMS + VAAPI via libdrm/libva FFI.
