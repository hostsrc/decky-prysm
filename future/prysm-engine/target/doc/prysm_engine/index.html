Prysm Engine — Low-latency screen streaming for Steam Deck

Architecture (MVP): FFmpeg (kmsgrab → VAAPI H.264) → RTP → prysm-engine → WebRTC → Browser

Architecture (v2 — native): PipeWire (Gamescope) → DMA-BUF → VAAPI encode → WebRTC → Browser
